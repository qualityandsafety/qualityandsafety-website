# Quality & Safety - Sito Web

Sito web istituzionale di Quality and Safety Srl.

## Descrizione

Questo repository contiene i file del sito web di Quality and Safety, inclusi:

- Pagine HTML del sito istituzionale
- Landing page per servizi (MarteWeb, MarteExe, RENTRI, Archiv.IO, GDPR)
- File CSS per lo stile del sito
- Immagini e logo
- Documentazione e certificazioni

## Struttura

- `index.html` - Homepage principale
- `marteRentri.html` - Landing page ottimizzata per Google Ads per il servizio RENTRI
- `MarteWeb.html` - Landing page per MarteWeb
- `marteEXE.html` - Landing page per MarteExe
- `Archiv.IO.html` - Landing page per Archiv.IO
- `gdpr.html` - Pagina GDPR
- `certificazioni.html` - Pagina certificazioni
- `stile.css` - Fogli di stile principali
- `web/` - Directory contenente applicazione Flutter

## Servizio RENTRI

La landing page `marteRentri.html` è ottimizzata per campagne Google Ads e include:
- Sezione prezzi trasparente (5€/anno + IVA per RENTRI, 15€/anno + IVA per conservazione)
- Call-to-action diretti verso la pagina di autenticazione
- Design responsive e orientato alle conversioni

## Tecnologie

- HTML5
- CSS3
- Bootstrap 4.3.1
- Font Awesome 5.15.4
- JavaScript/jQuery
